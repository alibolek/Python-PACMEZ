# PACMEZ

## What is this?

	Pacmez is basic 2D game created with python turtle library.

## How to play?

	* Download source code
	* Go to game directory
	* Open terminal here
	* Execute this command:

		` ~$ python3 pacmez.py `

	*Choose level difficulty
	*Enjoy it!
